## 🧠 About the Project

Moodify is an immersive web-based music experience designed to create a **personal emotional space through sound and visuals**.
It combines minimal UI, smooth interactions, and dynamic audio playback to deliver a calming and engaging environment.

Instead of just playing songs, Moodify focuses on **how music feels**.

---

## ✨ Features

* 🎵 Seamless music playback
* 🌈 Mood-based aesthetic design
* 🌀 Smooth and minimal UI
* ⚡ Lightweight and fast
* 🎧 Personal music space concept
* 📁 Organized local song management

---

---

## ⚙️ How It Works

* Songs are stored locally in the `/songs` folder
* JavaScript dynamically loads and plays audio
* UI responds to user interaction for playback (browser autoplay policies)
